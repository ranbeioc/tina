关于细项的产品需求和运营规则（具体方案待出）

一、针对用户的各处收入点拆解

```
1、新用户
a：加大赠送权益，在k歌上传，签到页面，首充，参加活动方面，适当拓宽免费以及折扣的权益。
b：社交方面新增好友，获得一定的权益。
2、老用户（活跃但不花钱）
a：分享得到免费权益。
b：聊天，赠免费花，点赞，评论，转发，均有相关的权益到帐。
c：活动参与相关权益（参与权益）
3、老用户（花钱）
a：高端用户权益
b：商城，宠物，个性化主页，头像框＋勋章，礼物，有声礼物，主页挂件
c：直播间收费喇叭（花钱滚屏公告）
d：活动冠军奖牌勋章（各种奖牌荣誉体系）
e：线下周边，合作周边（实体礼物）
f：道具库（活动参与卡，积分加速卡……）
g：首页展示（ugc作品可以上首页，排行榜相关）
4、会员系统（详见会员规划）
a：成长体系
b：会员权益
```

二、收入点细拆分说明

```
1、签到/抽奖
细化，新增签到功能，主推新用户进行签到，培养衷诚度。
转盘抽奖，每日一次免费的签到机会
2、付费小喇叭
直播间首页滚屏，付钻石即可购买，滚动播出相关信息。
3、活动配置多样化
活动门槛设置，需要道具方可以参加活动，需要提交钻石方可购买道具等。
单一礼物＋多种礼物的榜单配置榜单。
榜单支持多种榜单排列组合方式。（例如男性用户榜＋女性用户榜，限用户性别，可以得到配对头像冠军框）
活动支持上传用户手机号＋邮箱＋个人头像框，后台审核后，方可上传作品成功，前台展示。（配合市场合作推荐）
活动支持前台作品全部展示，只展示有礼物的作品，或者其它展示数量。（配合市场宣传的时候隐藏官方活动真正参与人数）
活动支持……（待全）
4、作品冠军曲库
限定冠军作品不能删除，历界冠军作品展示。
5、个性化推荐
用户喜好的歌曲同类新歌上线，推荐
用户最近打赏的用户的优质作品上线，推荐
用户唱的最多的歌曲的mv作品，推荐。
用户社交最多的用户的作品，推荐。
（以上需要讨论相关需求，再出细化方案，精细化推荐可以减少用户搜索时间，实现快速消费的目的）
6、礼物
大礼包促销：多款礼物，打折促销（现有是买钻石赠送k币礼物）
限时促销：例如消费低迷时间段，对某些情境礼物打折促销上量。
限xx抢购：跟现在的限时礼物很像，但要设定有些礼物，只有些人才能购买，限新用户，级别，男，女，地域……
带音效的礼物：带音乐，带歌曲，带高音部分……
7、头像框＋勋章库
增加动效头像框，例如，腾讯的
8、表情商城
会员专属，会员限时使用7天，其它限时上新。
9、ugh作品框
购买作品各种动效，静态框，彰显用户不同之处。
10、个性化主页＋挂件
主页变色，主页挂件需集齐相关礼物在主页显示。
11、道具库
活动参与卡，积分加速卡……
12、宠物系统
13、荣誉系统
冠军奖章证书，可分享。阶段性奖励奖章证书。需求需要具体出。
14、会员商城
会员特权里享受的相关权益
15、积分商城
同会员商城，用户的相关积分可以兑换线下商品。
16、合作周边礼物＋天籁周边礼物＋活动冠军线下小奖章
实体礼物展卖。（需要具体需求）
```

天籁精细化运营方向原则设定：

```
1､用户精细化

对用户进行基础信息收集和喜好标签收集。对用户进行简单的画像。

2､流量精细&匹配化

根据不同的地域，不同渠道，不同用户年龄层次……设计运营方案

3､数据精细&结果化

用户，活动，收入（ugc，直播间，会员，用户主页，商城，表情商城，个性化主页，道具卡，），用户行为数据

4､运营效率精细&自动化

根据用户画像，自动推送内容（最新活动，匹配用户，打打礼物，新内容上线……）

根据用户已被证实有效的行为，后续实现自动推送。 

制定不同的运营活动，礼物等相关策略，有针对性的对用户进行投放或者全面投放。
```
