# 审核组

访谈时间：2017-06-06 17:00

以下内容根据昨天会议内容整理后的各类问题，具体如下：

后台优化（优先级从高到低）：

```
1，增加直播视频流，单独点击一个直播视频，页面跳转至有声的实时视频直播页面，并且该页面含有本次视频直播的所有视频截帧
（此需求在视频直播改版以后）

2，增加直播视频流历史截图、音视频端口（邮件已批未做）

3，后台对账号封禁，前端立刻把该账号强制下线（目前技术组同事已经在解决中）

4，封禁直播间后，直播间立刻关闭，该房间内所有用户被踢出该房间

5，优化敏感词禁用逻辑，目前敏感词只能对连贯的文字、数字生效，对各类标点符号以及特殊字符无法生效，另外优化关联词逻辑，
例如关联名词为A 关联动词为B，目前逻辑为AB连贯出现才会生效，希望可以优化至发表内容中只要含有AB即可生效，例如发送内容
为“ACB”包含A\B即可生效

6，目前有用户下载我们平台曲库内容后，自行将MV替换成违规内容，增加MD5验证，与曲库核对MD5码，用户上传的音视频MD5与曲
库不匹配或自行导入的伴奏单独进入一个模块审核或者用颜色高亮显示

7，后台增加通过IP或设备号查询登录过的ID

8，音视频播放时增加多倍速播放按钮

9，修复敏感词库只能添加不能删除问题

10，用户头像上传时间不准确

11，后台增加对账号禁言功能，禁言后该用户可以正常浏览但不能发布任何评论或私信

12，前台家族ID显示、活动ID，在用户主页显示天籁号用户上传音视频后，如更改封面或修改介绍等操作，后台无法查询到更改后
的内容，并且希望优化单独删除用户音视频封面或介绍时给用户下发的私信，目前统一提示是删除音视频

13，增加被删除音视频作品播放模块，目前被删除的音视频作品无法播放

14，微信方式登录用户，删除昵称后，系统会自动同步微信昵称

15，前台家族ID显示、活动ID，在用户主页显示天籁号
```

前端：

```
1，运营版客户端增加管理权限，可直接从前台对用户警告、封号、禁言、踢出房间、封号、封房间等操作

2，具有管理功能账号不受房间限制，可隐身进入所有房间

3，房主或房间管理不具有对管理账号的处理功能

4，运营客户端增加举报反馈按钮，后台增加相应模块查看举报内容，解决听歌组或其他部门同事发现违规内容反馈繁琐或无渠道
反馈的问题
```

其他：

```
1，用户产生的UGC数据量过大，按照审核组目前人员编制只能对视频直播监控、音视频审核、用户上传相册审核、前台巡查，其
中由于音视频数据量过大，审核组无法全部审核完成，只能根据点播数排序将近半月左右播放数较高的内容审核通过，播放数较低
或无播放数的音视频、用户头像无法顾及，用户昵称、评论目前主要靠敏感词管控

2，垃圾广告过多的问题，通过两次功能优化，目前采取提取敏感词屏蔽、封禁该账号及对应设备并短期封禁IP的方式，可以有效
的控制数量，但无法彻底解决该问题

3，引入第三方智能审核可以在用户昵称、评论、垃圾广告、头像、相册这几个方面取得很好的效果，但智能审核对音频内容的审核
工作帮助不大

4，对于申请增加、优化的需求领导批复后，希望项目组能够回复排期邮件
```
